# ZKManager
简单权限控制下的zookeeper管理项目

# 步骤：
>1,使用maven构建项目
>2,修改 项目\src\main\resources\zk.properties 文件为可用配置
  userquery只能查询的用户<br>
  useradmin可修改数据的用户<br>
  h2con , h2user , h2password 修改为自身h2配置<br>
    可参考博客安装h2数据库：<br>
    http://blog.csdn.net/xvshu/article/details/51726291
